# Perintah
{: #command}

Ini adalah perintah dari module *showtimes* yang merupakan module utama dari bot ini

## Perintah Pengunjung
{: #command-user}

**!tagih \<judul>**<br>
Melihat status terakhir dari Judul yang diberikan, kalau gak nulis judul, akan di list semua garapan yang dikerjain
kalau misalkan dah tamat dikerjain akan muncul ``Garapan sudah selesai!`` atau semacamnya.
{: .notice}

**Contoh**:

>!tagih kyuuketsuki

**!jadwal**<br>
Melihat seluruh jadwal anime airing yang diambil sebuah Fansub.
{: .notice}

**!staff \<judul>**<br>
Melihat staff yang mengerjakan judul anime tertentu
{: .notice}

**Contoh**:

>!staff kyuuketsuki

## Perintah Staff
{: #command-staff}

**!beres \<posisi> \<judul>**<br>
Menandakan salah satu posisi untuk mengupdate status salah satu garapan
{: .notice}

**Contoh**:

>!beres tl kyuuketsuki

**judul**: bisa disingkat sesingkat mungkin ;)

**!gakjadi \<posisi> \<judul>**<br>
Ini kebalikan dari command **!beres**
{: .notice}

**Contoh**:

>!gakjadi tl hitoribocchi

**!rilis \<judul>**<br>
Merilis episode dari judul yang dikerjakan<br>
<br>
**!rilis batch \<jumlah> \<judul>**<br>
Merilis jumlah episode tertentu dari judul yang dikerjakan<br>
<br>
**!rilis semua \<judul>**<br>
Merilis semua episode dari judul yang dikerjakan
{: .notice}

**Contoh**:

>!rilis yuusha

>!rilis batch 3 yuusha

>!rilis semua yuusha

**Note:** Untuk perintah !rilis batch, terdapat \<jumlah> episode yang mau dirilis.
Penghitungannya adalah **Episode terakhir yang sedang dikerjakan** ditambah **jumlah**.<br>
Misalkan lagi **ngerjain Episode 4**, terus mau **rilis sampai episode 7**<br>
Total dari **Episode 4 sampai 7 ada 4** (4, 5, 6, dan 7)<br>
Maka tulis **jumlahnya 4**
{: .notice}


**!tandakan \<posisi> \<episode> \<judul>**<br>
Mengubah status posisi tertentu untuk episode tertentu dari belum ke sudah atau sebaliknya<br>
Modelnya seperti tombol on/off, hanya membutuhkan command yang sama untuk menandakan sudah atau belum<br>
Berguna untuk menandakan episode setelah atau yang jauh dari "episode sekarang" yang muncul kalau pakai command `!tagih`
{: notice}

**Contoh**:

>!tandakan ts 8 hitoribocchi

**\<posisi> ada 6, yaitu**:<br>
``tl``: Translator<br>
``tlc``: TLCer (Pemeriksa Terjemahan)<br>
``enc``: Encoder<br>
``ed``: Editor<br>
``tm``: Timer<br>
``ts``: Typesetting (Tata Rias)<br>
``qc``: Quality Check (Pemeriksa Akhir)
{: .notice}

## Perintah Admin
{: #command-admin}

**Note**: Admin ditambahkan manual oleh saya sendiri kedalam Database, jadi silakan PM saya untuk penambahan.
{: .notice}

Ada 2 command, yaitu:

**!tambahutang**<br>
Menambah anime/garapan kedalam database naoTimes<br>
Ini merupakan versi 2 dan pengembangan dari versi awal yang super panjang menjadi lebih simpel dan mudah<br>
Ikuti semua prosesnya dan atur sampai benar dan tepat baru masukan ke database
{: .notice}

**Ada settings yang bisa diubah dalam proses penambahan, yaitu:**

**1\. Samakan waktu tayang**<br>
Berguna untuk anime netflix sekali rilis banyak agar tidak terjadi kesalahan ketika waktu sisa sebelum tayangnya.

Akan ditambah lebih banyak lagi nanti ðŸ˜‰
{: .notice}

**Contoh**:

>!tambahutang

**!lupakanutang \<judul>**<br>
Drop utang/garapan dan delete dari database
{: .notice}

**Contoh**:

>!lupakanutang hitoribocchi

**!ubahstaff \<id_staff> \<posisi> \<judul>**<br>
Mengganti staff yang mengerjakan untuk sebuah garapan.<br>
**id_staff** adalah ID yang baru
{: .notice}

**Contoh**:

>!ubahstaff 0987654321 tl hitoribocchi

**!ubahrole \<id_role> \<judul>**<br>
Mengganti role yang mengerjakan sebuah garapan<br>
**id_role** adalah ID yang baru
{: .notice}

**Contoh**:

>!ubahrole 178649816238613 hitoribocchi

**!tambahepisode \<jumlah> \<judul>**<br>
Menambah episode dari total episode yang ada.<br>
**jumlah** merupakan total yang mau ditambah
{: .notice}

**Contoh**:

>!tambahepisode 4 hitoribocchi

>Menambah 4 episode dari total 12 episode (misal)

**!hapusepisode \<range> \<judul>**<br>
Menambah episode dari total episode yang ada.<br>
**range** merupakan ranged number, jadi bisa satu digit untuk episode itu saja.<br>
atau range dari episode x sampai y.
{: .notice}

**Contoh**:

>!hapusepisode 12 hitoribocchi (hapus episode 12)

>!hapusepisode 12-14 hitoribocchi (hapus episode 12 sampai 14)

**!supermotd**<br>
Membuat announcement dan mengirimnya ke semua admin yang terdaftar<br>
Tulis pesan lalu enter setelah itu cek sekali lagi dan pastikan<br>
sebelum mengirim ke semua admin.
{: .notice}

**Contoh**:

>!supermotd

## Perintah Owner
{: #command-owner}

Munculkan list perintah dengan `!ntadmin`

**!ntadmin tambah \<server id> \<admin id> \<progress_channel>**<br>
Menambah server baru ke database naoTimes.<br>
Bisa rujuk ke [Cara menambahkan server Discord Fansub](#tambah) untuk cara mendapatkannya.
{: .notice}

**Contoh**:

>!ntadmin tambah 1234567890 0987654321 1029384756

**!ntadmin hapus \<server id>**<br>
Menghapus server dari database naoTimes.<br>
**server id** merupakan ID server discordnya
{: .notice}

**Contoh**:

>!ntadmin hapus 1234567890

**!ntadmin tambahadmin \<server id> \<admin id>**<br>
Menambah admin ke server yang terdaftar di database naoTimes.<br>
**server id** merupakan ID server discordnya<br>
**admin id** merupakan ID discord orangnya
{: .notice}

**Contoh**:

>!ntadmin tambahadmin 1234567890 0987654321

**!ntadmin hapusadmin \<server id> \<admin id>**<br>
Menghapus admin dari server yang terdaftar di database naoTimes.<br>
**server id** merupakan ID server discordnya<br>
**admin id** merupakan ID discord orangnya
{: .notice}

**Contoh**:

>!ntadmin hapusadmin 1234567890 0987654321

**!ntadmin fetchdb**<br>
Mengambil database dari github dan mengirimnya ke discord.
{: .notice}

**Contoh**:

>!ntadmin fetchdb

**!ntadmin patchdb**<br>
Mengganti database dengan attachments yang dicantumkan dari discord<br>
Cantumkan file .json yang valid, lalu ketik `!ntadmin patchdb` sebagai pesannya baru enter.
{: .notice}

**Contoh**:

>!ntadmin patchdb

**!ntadmin forceupdate**<br>
Memaksa update database gist/utama dengan database local.
{: .notice}

**Contoh**:

>!ntadmin forceupdate

# Menambahkan ke server Discord Fansub 
{: #tambah}

1\. Tambahkan terlebih dahulu ke server: [Invite bot](https://discordapp.com/oauth2/authorize?client_id=558256913926848537&permissions=1544027248&scope=bot)<br>
2\. PM ``N4O#8868`` dengan format seperti ini:

```
Server ID: 
Admin ID 1:
Admin ID 2:
#progress announce Channel ID:
```

3\. Tunggu sampai dikonfirmasi oleh saya<br>
4\. Tambahkan utang dengan ``!tambahutang``

Jika ada episode lama yang ingin dihilangkan, silakan PM nama Animenya serta Episode yang mau dihapus
{: .notice}

Jika ada kesalahan atau ingin menambah admin, silakan PM saja
{: .notice}

>Cara mengambil Server ID yaitu dengan klik kanan logo servernya lalu klim Copy ID

>Cara ambil Admin ID yaitu klik kanan nama user lalu Copy ID

``#progress`` Channel adalah tempat setiap Update progress terjadi agar bisa dilihat pengunjung.<br>
Silakan buat channel dan berikan akses ``Send Message`` ke botnya<br>
Copy ID channelnya dengan cara klik kanan lalu Copy ID.
